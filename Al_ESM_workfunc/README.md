# Al_ESM_workfunc

This directory contains input files and scripts for calculating the work function of Al surfaces using slab calculations with Quantum ESPRESSO.

The slab structures are generated with ASE using the Python script:

```bash
python gen_Al_slab.py
```

The main target is the Al(111) surface. The slab-thickness dependence of the calculated work function should be checked by changing the number of Al layers.

## Purpose

The purpose of this calculation is to evaluate the work function of an Al slab.

The work function is calculated from

```text
Phi = Vvac - EF
```

where:

- `Phi` is the work function,
- `Vvac` is the vacuum level,
- `EF` is the Fermi energy.

In the present calculations, the Effective Screening Medium (ESM) method is used with boundary condition `bc1`. With this setting, the vacuum level is set to zero. Therefore, the work function can be obtained by checking the Fermi energy:

```text
Phi = -EF
```

when `Vvac = 0`.

## Slab generation

The Al slab structure is generated using ASE in `gen_Al_slab.py`.

The script constructs an Al(111) slab using `ase.build.fcc111`, adds vacuum along the surface-normal direction, centers the slab, and writes the Quantum ESPRESSO input file.

A typical slab-generation procedure is:

```python
from ase.build import fcc111
from ase.io import write

# FCC Al lattice constant in Angstrom
a = 4.044

# Al(111) slab with 6 atomic layers and 10 Angstrom vacuum
slab = fcc111(
    "Al",
    size=(1, 1, 6),
    a=a,
    vacuum=10.0,
)

# Center the slab around z = 0
z_positions = slab.positions[:, 2]
z_center = (z_positions.max() + z_positions.min()) / 2.0
slab.translate([0, 0, -z_center])

write("fcc111_6layer_slab.cif", slab)
```

The number of slab layers should be changed to examine the slab-thickness dependence of the work function.

## Quantum ESPRESSO setup

The Quantum ESPRESSO input file is generated by ASE using `format="espresso-in"`.

The calculation uses the ESM method:

```fortran
assume_isolated = 'esm'
esm_bc = 'bc1'
```

The important settings are:

```python
input_data = {
    "control": {
        "calculation": "scf",
        "restart_mode": "from_scratch",
        "prefix": "Al4",
        "outdir": "./tmp/",
    },
    "system": {
        "ibrav": 0,
        "ecutwfc": 40,
        "ecutrho": 400,
        "occupations": "smearing",
        "smearing": "mp",
        "degauss": 0.01,
        "assume_isolated": "esm",
        "esm_bc": "bc1",
        "nosym": True,
    },
    "electrons": {
        "conv_thr": 1.0e-8,
    },
}
```

The pseudopotential used in the calculation is:

```python
pseudopotentials = {
    "Al": "al_pbe_v1.uspp.F.UPF"
}
```

and the pseudopotential directory is specified by:

```python
pseudo_dir = "/yourdir/POT_GBRV/"
```

## Running the calculation

First, generate the slab and the Quantum ESPRESSO input file:

```bash
python gen_Al_slab.py
```

Then run Quantum ESPRESSO:

```bash
pw.x < Al_111_4_layers.in > Al_111_4_layers.out
```

or, on a parallel environment:

```bash
mpirun -np 32 pw.x < Al_111_4_layers.in > Al_111_4_layers.out
```

## Work-function evaluation

After the SCF calculation, check the Fermi energy in the Quantum ESPRESSO output file.

For example:

```bash
grep -i "Fermi" Al_111_4_layers.out
```

The general definition of the work function is:

```text
Phi = Vvac - EF
```

Since the ESM method with `bc1` sets the vacuum level to zero, the work function is obtained as:

```text
Phi = -EF
```

The result should be checked as a function of slab thickness.

## Potential profile

In addition to the Fermi energy, the electrostatic potential profile should be inspected.

Quantum ESPRESSO with ESM produces potential-profile files with names that include the calculation prefix and the `.esm1` extension.

For example, if the prefix is:

```text
Al6
```

then the corresponding ESM potential-profile file should be searched in the output directory, typically under `tmp/`, with a name containing:

```text
Al6.esm1
```

The potential profile in this file should be checked to confirm that the vacuum level is properly aligned and that the slab region and vacuum region are well separated.

## Thickness dependence

To confirm convergence of the calculated work function, repeat the calculation with different numbers of Al layers, for example:

```text
4 layers
6 layers
8 layers
10 layers
12 layers
```

For each slab thickness, extract:

1. the Fermi energy `EF`,
2. the work function `Phi = -EF`,
3. the potential profile from the `.esm1` file.

The final work function should be determined from the converged value with respect to slab thickness.

## Notes

- The slab should be centered along the surface-normal direction before writing the input file.
- A sufficient vacuum region should be used.
- The k-point sampling should be converged.
- The plane-wave cutoff energies should be checked for convergence.
- The potential profile in the `.esm1` file should always be inspected together with the Fermi energy.
